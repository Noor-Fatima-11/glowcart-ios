 💄 GlowCart – iOS Cosmetics Shopping App

**GlowCart** is a modern and elegant iOS UI design for an online cosmetics shopping experience.
It focuses on clean aesthetics, intuitive navigation, and a seamless user journey tailored for beauty product discovery and purchasing.

 ✨ Overview

This project showcases a mobile-first design approach for a cosmetics eCommerce app.
It highlights how thoughtful UI/UX can enhance user engagement and simplify online shopping.


 📱 Key Features

* 🟢 Smooth and engaging onboarding experience
* 🟢 Category-based product exploration
* 🟢 Clean and minimal home interface
* 🟢 Intuitive navigation flow
* 🟢 Visually appealing layout for product discovery


 🎨 UI/UX Highlights

* Soft and modern color palette (beauty-focused theme)
* Minimalist design with clear visual hierarchy
* Consistent spacing and typography
* User-centered design principles
* Optimized for mobile usability


🛠️ Tools & Technologies

* 🎨 Figma / Adobe XD (UI Design)
* 🖼️ PNG Assets
* 📱 iOS Design Guidelines


 🚀 Future Roadmap

* 🔹 Convert UI into a functional iOS app using SwiftUI
* 🔹 Implement product detail and shopping cart features
* 🔹 Add authentication (Login/Signup)
* 🔹 Integrate backend services (Firebase / REST API)
* 🔹 Enable secure payment gateway

---

🎯 Project Goal

To design a visually appealing and user-friendly cosmetics shopping app that delivers a premium digital shopping experience.
